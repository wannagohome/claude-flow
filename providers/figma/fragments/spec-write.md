## Figma Design Reference

{FIGMA_URL}

Use MCP tools (`figma-desktop`) to directly inspect the Figma design:
- Analyze screen structure and component hierarchy
- Identify UI elements present in design but not in input documents
- Check state variations (empty, loading, error, success)
- Record Figma nodeId for each screen in the spec

If no FIGMA_URL is configured, check if the user provided a Figma URL
in requirements-input.md or other input documents.
